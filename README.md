# baremetal-lab

このプロジェクトは、GitHub 上で CI を動かしたり、Self-hosted Runner を使って実機で動作確認を行ったりと、
そうした仕組みを学びたいという思いからスタートしました。

ただ、その前段として、CI でビルドやテストを行うための “しっかりした対象” を用意したくなり、
自分にとって馴染みのある組込みソフトウェアのベアメタル環境を一から整えることにしました。

ベアメタル環境のコードは ARM CMSIS.5.9.0 の構成を参照しつつ、
Microsoft Copilot との対話を通じて最小構成として再構築したものです。
GNU Make ベースのビルドシステム、スタートアップコード、リンカスクリプト、
そして C ランタイムの構成は、その対話の積み重ねによって形になりました。
（この README の文章も Copilot の提案をもとに整えています。）

プロジェクト構造や各レイヤの詳細は docs/ 以下にまとめています。
- [01_overview](docs/01_overview.md)
- [02_usage](docs/02_usage.md)
- [03_arch layer](docs/03_arch.md)
- [04_runtime layer](docs/04_runtime.md)
- [05_platform layer](docs/05_platform.md)

ビルド方法や実行方法については [02_usage](docs/02_usage.md) をご参照ください。

現在は QEMU lm3s6965evb を対象としています。

外部からの PR や Issue は受け付けていません。
